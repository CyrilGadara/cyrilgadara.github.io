Complete Guide to Arrays in JavaScript

1. What is an Array?

An array is like a list in JavaScript. It stores many items in one place.

Think of it like a shopping list:

const shoppingList = ['apples', 'milk', 'bread']

2. Creating Arrays

To make an array, use square brackets [ ]:

// Empty array
const emptyArray = []

// Array with items
const fruits = ['apple', 'banana', 'orange']

3. What Can Arrays Hold?

Arrays can hold different types of information:

// Array of words
const words = ['Hello', 'World', 'JavaScript']

// Array of numbers
const numbers = [1, 2, 3, 4, 5]

// Array of true/false values
const trueFalse = [true, false, true, true]

// Array of objects (things with properties)
const people = [
  {name: 'Alice', age: 30},
  {name: 'Bob', age: 25}
]

// Array of arrays
const matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]

Tip: To make long arrays easier to read, you can write each item on a new line:

const daysOfWeek = [
  'Monday',
  'Tuesday',
  'Wednesday',
  'Thursday',
  'Friday',
  'Saturday',
  'Sunday'
]

4. How Many Items in an Array?

Use .length to count items in an array:

const animals = ['cat', 'dog', 'fish', 'bird']
console.log(animals.length) // This will show: 4

5. Getting Items from an Array

To get an item, use its position number (index). Remember, counting starts at 0!

const colors = ['red', 'green', 'blue', 'yellow']

const firstColor = colors[0] // This is 'red'
const secondColor = colors[1] // This is 'green'
const lastColor = colors[colors.length - 1] // This is 'yellow'

If you ask for an item that doesn't exist, you get 'undefined':

const tenthColor = colors[9] // This is undefined

6. Changing Items in an Array

You can change an item like this:

const fruits = ['apple', 'banana', 'orange']
fruits[1] = 'grape'
console.log(fruits) // Now it's ['apple', 'grape', 'orange']

7. Array Methods

Here are some useful things you can do with arrays:

7.1 Finding an Item's Position

Use indexOf to find where an item is:

const pets = ['dog', 'cat', 'fish', 'hamster']
const posOfCat = pets.indexOf('cat') // This is 1
const posOfBird = pets.indexOf('bird') // This is -1 (not found)

7.2 Adding Items to an Array

Add to the start with unshift:

const numbers = [2, 3, 4]
numbers.unshift(1)
console.log(numbers) // Now it's [1, 2, 3, 4]

// Add multiple items
numbers.unshift(-1, 0)
console.log(numbers) // Now it's [-1, 0, 1, 2, 3, 4]

Add to the end with push:

const numbers = [1, 2, 3]
numbers.push(4)
console.log(numbers) // Now it's [1, 2, 3, 4]

// Add multiple items
numbers.push(5, 6)
console.log(numbers) // Now it's [1, 2, 3, 4, 5, 6]

7.3 Removing Items from an Array

Remove from the start with shift:

const fruits = ['apple', 'banana', 'orange']
const firstFruit = fruits.shift()
console.log(firstFruit) // This is 'apple'
console.log(fruits) // Now it's ['banana', 'orange']

// Remove multiple items
const secondFruit = fruits.shift()
console.log(fruits) // Now it's ['orange']

Remove from the end with pop:

const colors = ['red', 'green', 'blue']
const lastColor = colors.pop()
console.log(lastColor) // This is 'blue'
console.log(colors) // Now it's ['red', 'green']

// Remove multiple items
const secondLastColor = colors.pop()
console.log(colors) // Now it's ['red']

7.4 The Powerful splice Method

splice can add or remove items anywhere in the array:

// array.splice(start, deleteCount, item1, item2, ...)

// Adding items
const fruits = ['apple', 'orange', 'grape']
fruits.splice(1, 0, 'banana') // Add 'banana' at index 1
console.log(fruits) // Now it's ['apple', 'banana', 'orange', 'grape']

// Removing items
const numbers = [1, 2, 3, 4, 5]
const removed = numbers.splice(2, 2) // Remove 2 items starting at index 2
console.log(removed) // This is [3, 4]
console.log(numbers) // Now it's [1, 2, 5]

// Replacing items
const colors = ['red', 'green', 'blue']
colors.splice(1, 1, 'yellow') // Replace 1 item at index 1 with 'yellow'
console.log(colors) // Now it's ['red', 'yellow', 'blue']

8. Making a Copy of an Array

Use slice to make a copy:

const original = [1, 2, 3]
const copy = original.slice()
copy.push(4)
console.log(original) // Still [1, 2, 3]
console.log(copy) // Now [1, 2, 3, 4]
